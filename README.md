# homebrew-tap

Homebrew tap for [zarlcorp](https://github.com/zarlcorp) tools. Formulas are auto-updated by GoReleaser on each release.

## Usage

```
brew install zarlcorp/tap/zburn
```

Or tap first, then install:

```
brew tap zarlcorp/tap
brew install zburn
```
